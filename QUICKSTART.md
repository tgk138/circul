# Быстрый старт

## Самый простой способ запуска

### Windows

1. **Двойной клик** по файлу `start.bat` в проводнике Windows
   - Всё настроится и запустится автоматически!

2. **Или через командную строку:**
   ```cmd
   start.bat
   ```

3. **Или через PowerShell:**
   ```powershell
   .\start.bat
   ```
   ⚠️ **Важно:** В PowerShell всегда используйте `.\` перед именем скрипта!

### Первый запуск?

Если это первый раз, сначала запустите:
```bash
setup.bat
```
или в PowerShell:
```powershell
.\setup.bat
```

Это автоматически:
- Создаст виртуальное окружение
- Установит все зависимости
- Создаст файл `.env` (нужно будет вставить токен бота)

## Что нужно перед запуском

1. ✅ Python 3.10+ установлен
2. ✅ FFmpeg установлен и доступен в PATH (или указан путь в `config.py`)
3. ✅ Токен бота в файле `.env`:
   ```
   BOT_TOKEN=your_token_here
   ```

## После запуска

Бот будет работать и ждать сообщений в Telegram. Отправьте ему:
- Ссылку на видео (YouTube, Rutube, TikTok и т.д.)
- Или видео файл напрямую

Бот нарежет видео на кружочки по 10 секунд и отправит обратно!
